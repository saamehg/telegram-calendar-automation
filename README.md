# Telegram to Google Calendar Automation

## 📌 Overview
This project automates task creation by connecting a Telegram bot to Google Calendar.  
Instead of manually creating calendar events, users can simply send a message via Telegram, and the event is automatically added.

---

## ❗ Problem
Manually adding tasks to a calendar is time-consuming and interrupts workflow.  
It requires opening an app, filling fields, and managing details.

---

## 💡 Solution
I built an automated workflow that allows tasks to be created instantly via a Telegram bot.  
The system processes the message and creates an event in Google Calendar without manual input.

---

## ⚙️ How It Works
1. User sends a message to a Telegram bot  
2. The message is captured via Make.com  
3. The workflow processes the input  
4. A new event is created in Google Calendar  

---

## 🛠️ Tools & Technologies
- Make.com (automation workflow)
- Telegram Bot API
- Google Calendar API

---

## ✅ Results
- Reduced time required to create tasks  
- Simplified task management process  
- Enabled fast task input from mobile  

---

## 🔮 Future Improvements
- Add voice input using speech-to-text (LLM-based transcription)  
- Extract date/time automatically from natural language  
- Improve automation logic for smarter scheduling
